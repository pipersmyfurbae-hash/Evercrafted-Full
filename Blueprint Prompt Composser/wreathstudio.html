How it works
Blueprint in. Prompt out.
Blueprint to prompt
Map your wreath's materials, structure, and emotional profile to Midjourney-ready prompts automatically — no prompt engineering required.
Six prompt targets
Hero door shots, studio flat-lays, lifestyle porch scenes, close-up detail, editorial, and experimental — each with the right parameters baked in.
Fidelity control
Slide between exact blueprint replication and creative variation. Control --stylize, --chaos, and --sw for style references from one interface.

Quick tips
Fidelity 70–100 — enforces exact materials. Best for SKU-accurate product shots.
--sw 60–120 — balanced style reference. Keeps lighting consistent without overriding palette.
--chaos 0–10 — keeps outputs predictable for shop hero images. Raise for moodboards.
Experimental target — drops --style raw for concept exploration. Not for product images.
