How-To Guide

Design with math. Build with confidence.

This guide covers everything you need to generate precise, mathematically-balanced wreath blueprints — from your first click to exporting a Midjourney render prompt.

✦

Quick Start

You can generate your first blueprint in under 30 seconds. Here's how:

1

Select your wreath size

Choose Small (10"), Medium (16"), Large (24"), or Custom. Most door wreaths are 16–24 inches.

2

Choose a base type

Grapevine is best for natural, dried, or mixed designs. Pine/Wire suits full floral and ribbon-heavy wreaths.

3

Set your element counts

Use the ± counters to set Focal flowers (1–9), Filler/Greenery (1–20), and Accent elements (0–12). These drive the placement math.

4

Click Generate Blueprint

The engine runs instantly. You'll see a live SVG diagram, coordinate table, engine scores, and (Studio+) a Midjourney prompt.

Start with the Medium size + Golden angle spiral — this combination produces the most natural-looking, balanced designs for beginners.

✦

App Interface

Blueprint Generator uses a two-panel layout. The left panel contains all your design controls. The right panel shows your live blueprint output.

Left Panel (Controls)

  • Wreath Size — preset or custom diameter/depth
  • Base Type — Grapevine or Pine/Wire
  • Elements — Focal, Filler/Greenery, Accent counters
  • Placement Algorithm — Golden Angle, Radial, or Cluster
  • Emotion Input — optional text that influences scores and prompts
  • Generate button — runs the engine
  • Saved blueprints — your cloud-saved designs (Craft+)

Right Panel (Preview)

  • Engine Results — element counts + quality scores
  • Blueprint Name — editable; used in saves and exports
  • Placement Diagram — live SVG with stats row
  • Coordinate Table — (r, θ, x, y) for every element (Craft+)
  • Midjourney Prompt — auto-generated render prompt (Studio+)
  • AI Blueprint — describe → generate (Atelier)

Base Type

The base type doesn't affect the placement math, but it informs the Midjourney prompt and builder guide copy.

  • Grapevine — natural, visible texture between elements. Best for dried botanicals, neutral palettes, and rustic aesthetics.
  • Pine/Wire — dense, uniform. Best for full floral designs, ribbon work, holiday wreaths. Hides the base completely.

The three element categories map to different placement rings and visual layers of the wreath.

A good starting ratio: 3 focal : 8 filler : 5 accent. This follows the classic rule of odd numbers and creates natural visual weight distribution.

Placement Algorithms

The algorithm controls how elements are positioned around the wreath ring. Each produces a fundamentally different aesthetic.

✦ Default
Golden Angle Spiral
Each element placed at 137.5° from the previous — the golden angle from Fibonacci mathematics. Creates maximally even distribution with no repetition.
Best for: organic, natural, asymmetric designs
Uniform Radial
Elements spaced equally around the ring, grouped by type. Focal flowers evenly divided, fillers offset 15°, accents offset 30°.
Best for: geometric, formal, symmetrical designs
Cluster Grouping
Elements grouped in sets of 3 at 8 cardinal and diagonal points (0°, 45°, 90°… 315°). Spread ±22° around each anchor.
Best for: traditional, classic, holiday wreaths
✦

Placement Engine

The engine runs entirely client-side. When you click Generate, it:

  1. Calculates the usable radius range from your diameter (outerR = diameter/2 * 0.88)
  2. Assigns every element to a type-specific radius ring
  3. Applies the selected algorithm to distribute elements by angle
  4. Returns (r, θ) polar coordinates for every element
  5. Converts to (x, y) Cartesian for SVG rendering
  6. Calculates engine quality scores

SVG Diagram

The circular diagram renders live after every generation. It shows:

  • Outer ring — usable wreath boundary
  • Inner dead zone — the center boss (~20% radius) where no elements are placed
  • Dashed guide rings — at 45%, 70%, and 90% radius for orientation
  • Numbered element circles — color-coded by type, sized by prominence
  • Stats row — diameter, total stem count, density rating, algorithm name

Print the diagram using the PDF button. The browser print dialog renders the SVG cleanly at any scale.

The Emotion / Memory field is optional but powerful. What you enter influences:

  • The Emotion Fit score in Engine Results

Midjourney Render Prompts Studio

Studio and Atelier subscribers get an auto-generated Midjourney or DALL·E prompt every time they generate a blueprint.

What's in the Prompt

  • Wreath diameter and base type
  • Element counts and placement algorithm description
  • Season (detected from current date)
  • Emotion clause (if you entered an emotion input)
  • Default render settings: studio photography, white wall background, 4K, --ar 1:1 --v 6 --style raw

AI Blueprint Generation Atelier

The most powerful feature: describe your wreath in plain English and AI generates a complete blueprint — including element counts, placement algorithm, floral suggestions, emotion tags, and design notes.

Using "Generate from description"

1

Type your description

Be as specific or as vague as you like. Include florals, colors, size, season, mood, or occasion.

2

Click "Generate from description"

AI analyzes your description and returns a JSON blueprint. This usually takes 3–7 seconds.

3

Blueprint auto-applies

The AI's recommendations are instantly applied to all controls. The SVG and table regenerate automatically.

4

Review and adjust

The AI suggestion is a starting point. Adjust any control and regenerate to fine-tune the placement.

Using "Suggest elements"

Type a description and click Suggest elements to get 5–8 specific faux floral recommendations with type, color, and a rationale. AI will never suggest prohibited florals (cherry blossoms, pussy willow, twig-based blossom varieties).

AI generation requires an active internet connection and counts against your Anthropic API usage. Atelier tier includes unlimited calls. Generation may occasionally fail if the API is unavailable — retry in that case.

Save & Export

Placement Math Reference

Coordinate System

Wreaths use a polar coordinate system centered at the wreath's axis:

  • r — radius in inches from center (0 = center, r=diameter/2 = outer edge)
  • θ — angle in degrees (0° = right / 3 o'clock position, clockwise)

Converting to Cartesian (for SVG): x = r · cos(θ), y = r · sin(θ)

Golden Angle

GOLDEN_ANGLE = 137.507764°
theta[i] = (i × 137.507764) mod 360

This is derived from the golden ratio: 360° × (1 - 1/φ) where φ ≈ 1.618. The result is that no two consecutive placements land at the same angle — ever. It's the same pattern sunflower seeds use.

Radius Ring Formula

outerR      = (diameter / 2) × 0.88
innerDeadZone = outerR × 0.20

Focal ring:  rMin = outerR × 0.50, rMax = outerR × 0.80
Filler ring: rMin = outerR × 0.30, rMax = outerR × 0.95
Accent ring: rMin = outerR × 0.20, rMax = outerR × 0.88

FAQ

Can I change element counts after generating?

Yes — adjust any counter and click Generate again. The diagram updates instantly.

Why does my blueprint say "Emotion Fit: 73%"?

In Craft and Studio tiers, the Emotion Fit score is an approximation. In Atelier tier, the score is calculated by Claude API using your actual emotion input and the generated blueprint — making it accurate and meaningful.

What florals are prohibited?

Cherry blossoms, pussy willow, and twig-based blossom florals are permanently prohibited in all Evercrafted designs and AI suggestions. This is a canonical platform rule.

My AI generation returned an error. What do I do?

Wait 10–15 seconds and try again. If it persists, check your internet connection. Occasionally the Claude API experiences brief delays. The error is displayed in the AI output area.

How do I print just the SVG diagram?

Click PDF — the browser will print the full page but the blueprint card and diagram render cleanly. For diagram-only, right-click the SVG and save/copy from your browser.

What is the WGS genome string?

The Wreath Genome String is a compact, pipe-delimited representation of a complete blueprint. Example: 24|GV|F3|L8|A5|GLD|.... It lets you save, share, and reload any design as a single string. Available in Studio and Atelier tiers.
