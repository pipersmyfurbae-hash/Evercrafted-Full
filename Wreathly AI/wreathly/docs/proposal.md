## Introduction

WreathForge™ is the core AI-powered design engine within the WreathlyAI platform — a SaaS solution built for wreath designers, floral artists, craft entrepreneurs, and hobbyists who want to create, customize, and commercialize stunning wreaths at scale.

WreathForge™ transforms a blank canvas into a fully realized wreath design in seconds using generative AI, a drag-and-drop element library, seasonal theme intelligence, and real-time 3D preview. It bridges the gap between creative inspiration and sellable product — making professional wreath design accessible to anyone, and exponentially faster for experts.

**Target Users:**
- Independent wreath artists and Etsy sellers
- Floral design studios and event decorators
- Craft supply retailers offering custom design services
- Holiday decor brands building private-label product lines

**Market Opportunity:**
The wreath and seasonal decor market exceeds $4.2B annually in North America alone, yet no dedicated AI-powered design tool exists for this niche. WreathForge™ fills that gap.

## Design Elements

### Visual Language
- **Style:** Organic-luxe. Nature-inspired textures meet premium UI precision.
- **Color Palette:** Deep forest green (#1B4332), warm ivory (#FAF3E0), dusty rose (#D4A5A5), aged gold (#C9A84C)
- **Typography:** Display — *Cormorant Garamond* (elegant, editorial); Body — *DM Sans* (clean, readable)
- **Iconography:** Hand-drawn botanical line art; no generic icon packs

### Design System Components
- **WreathCanvas™** — The central circular design workspace. Elements snap to ring tracks.
- **ElementLibrary™** — Categorized panel: Greenery, Florals, Ribbons, Ornaments, Accents, Seasonal
- **ColorHarmony™** — AI-suggested palettes based on season, mood, or uploaded brand colors
- **StyleMoods™** — One-click aesthetic presets: Rustic Farmhouse, Nordic Minimal, Southern Magnolia, Coastal Boho, Urban Luxe, Holiday Classic
- **Ring Builder™** — Concentric ring layer system (inner, mid, outer, accent) for structural design
- **3D PreviewPane™** — Real-time rendered preview with ambient lighting simulation

### Wreath Structure Framework
Every WreathForge™ design is built on a parametric ring model:
1. **Base Ring** — Frame shape, diameter, material (grapevine, wire, foam, moss)
2. **Greenery Layer** — Fill foliage (eucalyptus, boxwood, pine, magnolia)
3. **Focal Layer** — Statement flowers or ornaments (3–5 focal points)
4. **Accent Layer** — Berries, pinecones, ribbons, picks
5. **Finishing Layer** — Bow placement, tags, twine, seasonal elements

## UI Overview

### Layout Philosophy
Three-panel horizontal layout:
- **Left Panel** — Element Library + AI Suggestions (280px, collapsible)
- **Center Stage** — WreathCanvas™ (flexible, zoom 50–200%)
- **Right Panel** — Properties + Layers + Export (320px, collapsible)

### Key UI Screens
1. **Onboarding Wizard** — Style quiz (5 questions) → AI-generated starter design
2. **Design Studio** — Primary workspace (described above)
3. **AI Prompt Bar** — Natural language input: "Make it more rustic with dried citrus"
4. **Variation Generator** — One click spawns 4 design variations
5. **Cost Estimator** — Real-time material cost calculator per design
6. **Export Suite** — PNG, SVG, PDF cut-sheet, Etsy listing image pack

### Accessibility
- WCAG 2.1 AA compliant
- Keyboard-navigable canvas
- Screen-reader labels on all elements
- High-contrast mode toggle
