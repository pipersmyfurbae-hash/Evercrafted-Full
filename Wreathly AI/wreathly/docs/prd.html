Section 01

Executive Summary & Problem Statement

WreathForge™ is the AI-powered design engine at the heart of WreathlyAI — a specialized SaaS platform for wreath designers, floral artists, craft entrepreneurs, and seasonal decor brands.

Today, wreath designers rely on a fragmented stack: Pinterest for inspiration, Canva for mockups, Excel for material costing, and third-party photo editors for product images. There is no purpose-built tool for the $4.2B wreath and seasonal wreath market.

Core Problem: Professional wreath design takes 4–8 hours per piece when done digitally. Beginner sellers can't visualize their ideas before spending on materials. No tool exists that combines AI generation, cost calculation, and direct marketplace export in one workflow.

WreathForge™ solves this by collapsing the design-to-listing workflow from hours to minutes, making professional results accessible to everyone from first-time crafters to wholesale studio managers.

Section 03

User Personas

Persona 1 — The Etsy Artisan ("Maya")

Maya runs a seasonal wreath shop with 200+ sales. She designs 15–20 new wreaths per season. Her pain: design time is her biggest bottleneck. She needs fast iteration, sellable mockups, and cost tracking.

Persona 2 — The Floral Studio Owner ("Rachel")

Rachel manages a 4-person design studio doing corporate and wedding installs. She needs brand consistency, team collaboration, and client-facing presentations. She values professionalism over speed.

Persona 3 — The Hobbyist ("Janet")

Janet makes wreaths for friends and local markets. She's not tech-savvy but wants to see her ideas realized before buying supplies. She needs simplicity, inspiration, and low commitment.

Section 04

User Stories

As an Etsy Seller

I want to generate 4 wreath variations from one prompt so I can test different aesthetics before committing to materials.

As a Studio Owner

I want to lock my brand color palette so every designer on my team produces on-brand outputs without manual checking.

As a Hobbyist

I want a 5-question style quiz that gives me a starting design so I don't face a blank canvas.

As a Seller

I want real-time material cost estimates so I know my margin before I list a product.

As a Designer

I want to export a ready-made Etsy listing image pack so I don't need Photoshop for product photos.

Section 07

Technical Architecture

Frontend

React 18 + TypeScript. Canvas: Fabric.js wrapped in custom WreathCanvas™ abstraction layer. Ring snap logic: custom polar coordinate grid system. Real-time state: Zustand. Animations: Framer Motion.

Backend

Node.js / Fastify API gateway. AI generation: Claude API (design interpretation) + Stable Diffusion fine-tuned on botanical assets. Element rendering: SVG-based with CSS transform composition.

AI Pipeline

User prompt → Claude (intent parsing + element selection JSON) → SVG composition engine → optional SD refinement pass → WebGL preview render.

Data

PostgreSQL (user data, designs, billing). S3-compatible object storage (design files, element library). Redis (session state, canvas sync).

Integrations

Etsy API v3, Shopify Partner API, Stripe (billing), Canva Connect API, QuickBooks Online API.

Section 08

Out of Scope

The following will not be built for MVP: physical material procurement/ordering, AR try-on (door preview), video content generation, mobile native apps (web-responsive only), print-on-demand fulfillment.
