Tab 1 Studio Generate prompts for a single wreath — explore, lock, catalog
Phase 1 — Find Your Direction
1
Configure your wreath specs

Fill in season, diameter, base type, form, composition, palette, foliage, florals, and accents. Or drop a feeling into the Emotional Brief banner and hit Configure — AI fills every field from the mood you described.

2
Leave Seed blank — click Generate 3 Variants — run Variant 1

Variant 1 is text-only, high chaos. This is your exploration pass. Paste it into Midjourney, evaluate all four outputs. You're looking for a direction, not a finished product. Expect 3–6 runs before something lands.

No seed on the first run. It only matters after you've found a direction you love.
3
Use Vary Subtle when you're close

If an output is almost right — composition is good but palette drifted, or florals read right but light is off — use Vary Subtle in Midjourney. 2–3 passes max. If it still isn't landing, go back to fresh generation.

Avoid Vary Strong early — it drifts significantly and can undo composition work you've found.
Phase 2 — Lock Your Reference
4
Copy the URL of the specific image you love

In Midjourney, click the individual image (not the grid), then copy its direct URL. This becomes your layout reference — it carries the exact compositional structure from that generation.

Copy the single image URL, not the job URL. Click the image first, then copy from the address bar or right-click.
5
Paste into Layout Ref URL — switch to Layout Lock — run Variant 3

Paste the URL into Layout Ref URL. Click the Layout Lock tab in Parameter Tuning and hit Apply Layout Lock Defaults. Then Generate 3 Variants again — use Variant 3 only. It applies your reference with locked parameters. The output should hold the structure of your reference while using your palette and florals.

Phase 3 — Production Catalog
6
Run the 4-Shot Catalog Set

When Variant 3 looks right, click 4-Shot Catalog Set. This generates four prompts — hero front view, front door lifestyle, close-up detail, angled editorial — all inheriting your layout reference. Run all four and evaluate as a set.

Shot 1 & 4 — most consistent. Primary product photos. Shot 2 — mood shifts slightly with the new background but the wreath holds. Shot 3 — composition irrelevant here. Purely about material and texture quality.
Tab 2 Collection Builder Design 3–6 coordinated wreaths from a brief or a story
1
Name your collection and choose a season and size

Give the collection a name — it flows into the lookbook title, the save file, and the assembly animation. Choose 3–6 wreaths. Optionally paste a shared layout reference URL if you want all prompts to inherit a compositional reference from the start.

2
Choose your brief mode — Standard or Story

Standard mode: Describe the collection feeling, season, any florals that inspire you. Claude designs the full palette, floral hierarchy, and composition variation using color theory and seasonal logic.

Story Mode: Give a memory, a place, a moment — Claude writes a poetic narrative first, then reads it like a design brief. Every sensory detail becomes a decision: light quality → palette temperature, textures → material choices, emotional arc → compositional progression. Each wreath becomes a chapter.

Story Mode produces the most cohesive collections. The story also carries forward into Lifestyle Prompts and the Lookbook.
3
Generate — then name each wreath

Claude designs all wreaths at once and fills every slot with a name, focal flowers, accent color, and two prompts — Explore (high chaos, no reference) and Production Lock (with reference + seed). The wreath name is editable — click it and type yours. The name embeds into both prompts immediately.

Name your wreaths before running them in Midjourney. The name becomes part of the prompt identity.
4
Run Explore prompts first — then lock with seeds

Copy each Phase 1 — Explore prompt into Midjourney. Find an output you love for each wreath. Copy that image's seed (from the Midjourney job details) and paste it into the slot's Seed field. The Production Lock prompt rebuilds automatically with the seed locked in.

5
Upload your final renders

Once you're happy with a wreath render, upload it to the slot using the Final Wreath upload zone. This image flows into the Lookbook gallery card and the assembly animation. Upload all wreaths before moving to Stage 2.

Tab 3 Lifestyle Prompts Place your wreaths in the world they belong to
1
Come here after your collection is generated

The status bar shows whether your collection story is loaded. If you used Story Mode, Claude already has the narrative it needs. If you used Standard mode, it works from the collection brief and wreath names.

You need a collection story for the best results. Story Mode collections produce the most atmospheric lifestyle prompts — the light quality, palette, and textures come directly from the narrative.
2
Click Generate Scene Prompts

Claude re-reads your collection story and designs six scenes: Front Door, Entry Hall, Living Room, Kitchen Window, Garden Gate, and Detail Shot. Each prompt places your wreath in that environment with light, surfaces, and mood derived directly from the story world.

3
Copy each prompt → run in Midjourney → upload the render

Use the Copy button on each card. Run in Midjourney. When you have a render you like, upload it to the card using the upload zone below the prompt. The image automatically flows into the Lookbook lifestyle gallery — no extra steps.

Use ✦ Refine on any card to deepen the light specificity and surface detail without touching the Midjourney parameters.
4
Shared seed — optional cohesion tool

If you want all six scenes to share a consistent mood and light quality, paste a seed into the Shared Seed field before generating. Leave blank to let each scene find its own expression.

Tab 4 ✦ Lookbook Generate the customer-facing page — ready to embed or share
1
Check the readiness checklist

The three indicators at the top of the tab tell you what's in place: collection name & story, wreath renders, and lifestyle renders. You can generate a lookbook at any stage — missing renders just leave those slots empty. Come back and regenerate after adding more.

2
Fill in brand settings

Set your brand name, CTA button text (e.g. "Shop This Wreath"), CTA link (your shop URL or leave as #), and accent colour. The colour picker and hex field sync both ways.

3
Click Generate Lookbook

Claude writes a lyrical one-sentence caption for each wreath drawn from your collection story, then assembles the full HTML document. The lookbook includes: a cinematic hero with your story, animated assembly sequence, wreath gallery with captions and CTAs, lifestyle scene gallery, and a branded footer.

4
Preview — then Download

Preview opens it in a new tab instantly so you can scroll through and evaluate. Download HTML saves a standalone file named after your collection. Drop it directly into your website, host it on any static server, or send it as a link. It has no external dependencies — everything is self-contained.

The lookbook is a complete standalone HTML file. Your customer never sees the studio — just the finished editorial page.
⬇ Save & Sessions Protect your work — save to any drive, restore any time
✦
Save to file — works on any drive

The ⬇ Save button in the header exports your entire session as a .json file named after your collection. Save it to a thumb drive, Dropbox, a folder per collection — wherever you want. Everything is in the file: collection name, story, all prompts, seeds, render images (as base64), lifestyle prompts, lifestyle renders, and lookbook settings.

✦
Load any session — pick up exactly where you left off

The ⬆ Load button opens a file picker. Select any session file and the full app state restores instantly — every field, every prompt, every image.

✦
Autosave — your safety net

Every change triggers a 3-second autosave to your browser's local storage. The header shows ● unsaved changes in amber while it's pending, then ✓ saved in green when it writes. If you close the tab accidentally and reopen the app, a banner appears with the collection name and time — one click restores everything.

Autosave is a safety net, not a backup. Always save to file before closing the browser or clearing browser data.
✦
New Collection — clear with confidence

The ✕ New button opens a confirmation before clearing anything. It clears the collection state only — Studio settings, lookbook branding, and your theme stay. Save to file first if you want to keep the current collection.

Notes & Tips
✦
✦ AI Refine on any prompt

Every prompt — Studio variants, Collection slots, Lifestyle scenes — has an ✦ AI Refine button. It deepens emotional language, sharpens clock-position descriptions, and adds material texture specificity, without touching any Midjourney parameters.

✦
Discord mode for --sref

The --sref style reference parameter is Discord-only — not available in the Midjourney web UI. Enable Discord Mode in Photography & Setting only when working from the Discord bot. The app hides those fields by default.

✦
Reusing a layout reference across a collection

Once you find a reference image that consistently produces the right compositional structure, it can anchor your whole collection. Set it in the Collection Ref URL field in the Collection Builder — all Production Lock prompts inherit it. When palette and florals change between wreaths, the structure stays consistent.

✦
The pipeline is non-linear

You don't have to follow the tabs in order every time. Start a Studio run to test a palette before committing to a collection. Generate a lookbook mid-collection to see how it's looking. Upload lifestyle renders one at a time as you get them. The session autosaves every step.
