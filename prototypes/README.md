# Prototypes (archived — not part of the live app)

Earlier standalone iterations of the blueprint engine, kept for reference.
They use a hardcoded demo inventory and are **not** wired to the server.

- `evercrafted-blueprint-engine.html` — original engine (v1)
- `evercrafted-unified-engine.html` — v1 + formula selector
- `emotion_formula_selector.html` — standalone formula-picker demo

The production engine is the server-wired **memory-scene** (memory-driven) and
**v2-engine** (manual design). Both design from your real Supabase inventory.
